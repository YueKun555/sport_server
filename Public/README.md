# 使用教程

使用手机浏览器打开链接，不要在微信里打开。

## 安装App

### 1. __下载快乐运动[点击打开链接，点击开始运动下载](https://sport.yuekun.net)，注册并登录。__

### 2. __下载积木编程[点击打开链接，点击下载客户端下载](https://www.touchsprite.com/luabox)，注册并登录。__
